#FLASK BLOG
$ pip install flask

Now set environnment:

-for Mac/Linux :
$ export FLASK_APP=myfile.py

-for Windows : 
$ set FLASK_APP=myfile.py

For hot reload you must put on the debug mode (export for Mac / Linux and set for Windows).

$ set FLASK_DEBUG=1

Then launch app with 
$ python myfile.py
